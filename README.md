# Estudio A Web — versión optimizada

Estructura lista para GitHub Pages, Netlify o cualquier hosting estático.

## Estructura
- `index.html`
- `assets/css/styles.css`
- `assets/js/main.js`
- `assets/img/...`
- `assets/logos/...`

## Publicación en GitHub Pages
1. Subir todo el contenido del proyecto al repositorio.
2. Verificar que `index.html` esté en la raíz.
3. Activar `Settings > Pages > Deploy from a branch > main > /(root)`.
